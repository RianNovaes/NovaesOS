# NovaesOS

Sistema operacional educacional desenvolvido em Assembly x86.

Objetivos:
- Aprender bootloaders
- Aprender gerenciamento de memória
- Aprender interrupções
- Aprender arquitetura de computadores

Status:
[Concluído] Bootloader
[Em andamento] Terminal
[Planejado] Sistema de arquivos
[Planejado] Multitarefa
